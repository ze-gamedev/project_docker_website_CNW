instalando os últimos pacotes

Com o repositório configurado, o usuário pode decidir entre instalar a última versão ou uma versão específica.


Caso deseje instalar a última versão:

sudo yum install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin



Caso deseje uma versão específica

yum list docker-ce --showduplicates | sort -r


O comando acima deve resultar em

docker-ce.x86_64 3:28.3.3-1.el9 docker-ce-stable
docker-ce.x86_64 3:28.3.2-1.el9 docker-ce-stable
...


Baseado na saída adquirida, escolha sua versão

VERSION_STRING=28.3.3-1.el9
sudo yum install -y docker-ce-$VERSION_STRING docker-ce-cli-$VERSION_STRING containerd.io docker-buildx-plugin docker-compose-plugin
