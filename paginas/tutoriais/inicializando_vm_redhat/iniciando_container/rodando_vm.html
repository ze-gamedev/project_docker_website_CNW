rodar uma vm containerizada (exemplo com ubuntu)

Um container é como uma VM leve. Pra iniciar um container interativo com Ubuntu:

sudo docker run -it ubuntu bash


docker run - cria e inicia o container.
-it - deixa interativo, com terminal.
ubuntu - imagem usada (baixa do Docker Hub se não tiver).
bash - comando que vai rodar dentro do container.
