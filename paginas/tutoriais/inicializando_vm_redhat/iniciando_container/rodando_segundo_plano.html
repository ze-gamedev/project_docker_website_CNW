rodar em segundo plano (modo daemon)

Se quiser subir a “VM” sem entrar nela:

sudo docker run -d --name minha_vm ubuntu sleep infinity


-d → detached (roda em segundo plano).
--name minha_vm → dá um nome pro container.
ubuntu sleep infinity → roda o Ubuntu e mantém ativo com um processo “infinito”.


Após entrar na VM:


sudo docker exec -it minha_vm bash
